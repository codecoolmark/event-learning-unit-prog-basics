Events around us

Introduction

Events are all around us. Birthday, Celebarations, Christmas, New Year etc. Interestingly any interaction with a computer is also based on events. Every time we click on a mouse or press the enter key or really any key an event is generated. If we want to build interactive applications we need a way to react to these kinds of events in our code.
